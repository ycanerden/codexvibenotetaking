# Granola-Style Notes (Web MVP)

## Run in 2 steps
1. Double-click `Start.command` (it opens a local web server on port 5177)
2. Open `http://localhost:5177` in Chrome

## Recording notes
- You must choose **Chrome Tab** and enable **Share tab audio**
- Works best when the Google Meet call is in the shared tab

## API
- Paste your Groq API key in the UI
- You can change transcription or chat model names at any time
